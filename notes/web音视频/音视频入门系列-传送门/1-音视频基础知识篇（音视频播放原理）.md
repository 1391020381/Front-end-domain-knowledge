# 音视频基础知识篇（音视频播放原理）

* 音视频播放的原理主要分为
    - 解协议
    - 解封装
    - 解码
    - 音视频同步
    - 播放


1. 流媒体数据
2. 解协议  RTMP HLS HTTP-FLV
3. 封装格式数据   AVI FLV TS
4. 解封装
    - 音频压缩数据 AAC MP3
    - 音频解码 FFmpeg
    - 音频原始数据  pcm 

    - 视频压缩数据 H.264
    - 视频解码  FFmeg
    - 视频原始数据  YUV RGB   


    - 音视频同步

    - 播放音频  SDL
    - 播放视频  SDL
